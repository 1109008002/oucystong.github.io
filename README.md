# 怪兽导航

## 开始之前

1、浏览器的收藏夹内容原来越多，有时候无法很快的找到一个不常用的网址，所以萌生了一个想法：搞一个自己的个人导航页。

2、如果你跟我有一样的需求，那么可以试试怪兽导航，部署方便简单，不需要任何服务器成本，可以说是：零成本。

3、唯一需要的能力就是会简单使用Git、Github和简单的修改前端代码。
